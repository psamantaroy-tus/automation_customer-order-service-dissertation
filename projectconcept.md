Project Title:	Design Implement and Evaluative – automated cloud infrastructure deployment using Terraform
Project Background:	At present scenarios companies are doing the cloud provisioning through traditional manual effort, which is time taking, error prone and longer to execute and  also there are environment inconsistency. Using automation from provisioning infrastructure to deployment of services will help to reduce deployment timing which will resolve the environment inconsistency , so will be easier to replicate same environment and rebuild many times with much less error. 
Project Challenge:	Automate the whole process of deployment from cloud infrastructure provisioning to deployment of services in the systems. Different tools can be used in order to implement the end-to-end automation – aws for cloud infrastructure, Terraform, docker, GitHub action for CI/CD

Proposed Approach:	Different technologies will be using for the automated infrastructure deployment. Terraform will help to write script to provisioning the cloud infrastructure by setting up instances, network, IAM role, S3 bucket and security, GitHub actions can be used to trigger the repo of the services and run the CI/CD workflows for microservice deplyment. Docker will be used to containerize the services.
